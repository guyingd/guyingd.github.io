添加友链页面需要先执行hugo new link.md然后编辑link.md在Front Matter中添加layout: link

在/data/目录下新建一个link.yml，HUGO支持三种格式yml,json,toml，保证名称为link即可

添加友链，编辑link.yml添加如下数据:

- { "name": "", "link": "", "description": "", "avatar": "" }

name 站点名称(必填)

link 站点地址(必填)

description 站点描述

avatar 站点头像
